# uniapp-peiyin
文字转人声配音uniapp多端支持的小程序
### 应用场景
短视频制作泛滥的今天，基于大多数人不想真人发声，此时配音小程序就可以一显身手了。 输入文案后自动生成mp3格式文件。保存到本地，结合剪映放的导入音频功能就能完美的为自己的小视频配上软妹子或者其他好听的的声音
